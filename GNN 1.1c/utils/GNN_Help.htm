GRAPH NEURAL NETWORK HELP
In several applications the information is naturally represented by graphs. Traditional approaches cope with graphical data structures using a preprocessing phase which transforms the graphs into a set of flat vectors. However, in this way, important topological information may be lost and the achieved results may heavily depend on the preprocessing stage. The new model proposed, called GNN (Graph Neural Network) extends recursive neural networks and can be applied on most of the practically useful kinds of graphs, including directed, undirected, labelled and cyclic graphs. More information on this model and on its learning algorithm can be found in [1]. A graphical demonstration of the tool is available in the Help/View the Demo section of the ConfigurationTool menu.

Neural model part contains, called Jacobian Control, Saturation Control, Transition Net and Output Net. On the contrary, Linear model part is divided into Transition Net, Saturation Control, Forcing Net and Output Net. In the following, detailed explanations will be given on all the sections.

Training Results Panel

In this panel the tool shows the results while the training is in progress and at the end of learning phase.
  1. Section Toolbar
    2. Plot area
    Results    		 In this area various kind of plots are shown, depending on the problem and on the selected value in the popup menu Plot type
    Popup menu
    Plot type    		 It allows to change the plot displayed in the Plot Area Results.
    If Training and validation errors is selected, the area shows the plot of this errors.
    If Stability Coefficient History is selected, the area shows the plot of this coefficient during learning phase.
    If Jacobian History is selected, the area shows the value of the Jacobian norm in the various epochs.
    If Jacobian Error History is selected, the area shows when the value of the Jacobian norm has become larger than the specified Jacobian threshold.
    If Number of forward iteration is selected, the area shows how many forward iteration the tool has performed before reaching the stop coefficient or the maximum number of steps.
    If Number of backward iteration is selected, the area shows how many backward iteration the tool has performed before reaching the stop coefficient or the maximum number of steps.
    If Saturation Coefficient History is selected, the area shows the plot of this coefficient for the various net during learning phase.
    If testing results is selected (this option is available only with some problems), the plot area displays testing results.
    Button Plot all graphics detached It opens each plot available for the current results in a different Matlab figure
    Button Grid It toggles the grid on and off in the current plot
    Button Zoom It allows the user to zoom in and out in the current plot


  2. Buttons

    3. Button
    More learning         		This button continues learning
    Button Reconfigure Discard training results and reopen main panel for a new session
    Button Quit Quits the panel and the tool
    Button Test This button opens a new panel that shows testing results
